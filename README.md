### hello world - a basic electron project
#### hello world
```
hello-world/
├── web/                // 
    ├──css
    ├──js
    └──index.html
└── main.js             //
```

#### Steps
1. `npm init`,generate file package.json

2. add to package.json
```
  "devDependencies": {
    "electron": "^1.8.4",
    "robotjs": "^0.5.1"
  }
```

3. `npm install`

4. `electron .`
